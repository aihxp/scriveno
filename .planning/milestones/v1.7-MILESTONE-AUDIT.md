## Integration Findings

### Critical Gap

**FLOW-1: Non-sacred print builds fail to set TYPST_TEMPLATE**

- **File:** `commands/scr/build-print.md`, line 139
- **Symptom:** When `tradition:` is absent or null (all non-sacred projects), STEP 1.7 skip instruction routes to STEP 2, bypassing STEP 1.8. TYPST_TEMPLATE is never assigned. STEP 4's Pandoc invocation uses `--template={TYPST_TEMPLATE}` which expands to empty string.
- **Affected:** All prose, visual, poetry work type PDF builds — novel, picture book, chapbook, screenplay (when available)
- **Fix:** `build-print.md` line 139: `"proceed to STEP 2"` → `"proceed to STEP 1.8"`

### Medium Gap

**MISSING-1: stage_play work type cannot reach build-print**

- **File:** `data/CONSTRAINTS.json`
- **Symptom:** `stage_play` (script group) returns "not available for this work type" at STEP 1. scriveno-stageplay.typst route in STEP 1.8 is unreachable.
- **Fix:** Add `"script"` to `exports.build_print.available` and `commands["build-print"].available`
